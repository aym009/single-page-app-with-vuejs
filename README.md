# single-page-app-with-vuejs
Single page application based on VueJs 3 with [TVMaze api](http://www.tvmaze.com/api).

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run unit test
```
npm run test:unit
```
